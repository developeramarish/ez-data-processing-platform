# EZ Data Processing Platform

Enterprise data ingestion, validation, and monitoring system with React frontend and .NET microservices backend.

## Frontend

React-based frontend with TypeScript, Ant Design, and RTL (Hebrew) support.

### Features
- Data source management
- Schema validation
- Metrics configuration
- Real-time monitoring
- AI Assistant
- Multilingual support (Hebrew/English)

### Technology Stack
- React 19
- TypeScript
- Ant Design
- React Query
- React Router
- Monaco Editor
- SignalR for real-time updates

## Getting Started

### Frontend

```bash
cd Frontend
npm install
npm start
```

The application will run on http://localhost:3000
